该功能用于删除日志表。

**删除**

1. 在左侧导航栏，选择 **_"系统配置/日志审计/日志管理"_** 菜单项，进入日志管理页面。
2. 单击日志表右侧操作列 **_"删除"_** 按钮，弹出操作确认对话框。
3. 单击 **_"确定"_** 按钮，完成操作。

**批量删除**

1. 在左侧导航栏，选择 **_"系统配置/日志审计/日志管理"_** 菜单项，进入日志管理页面。
2. 在列表中选择一个或多个日志表，单击列表上方 **_"删除"_** 按钮，弹出操作确认对话框。
3. 单击 **_"确定"_** 按钮，完成操作。
