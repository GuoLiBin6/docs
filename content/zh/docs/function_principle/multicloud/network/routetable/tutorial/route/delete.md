title: "删除路由"
date: 2022-01-04T14:15:00+08:00
weight: 30
description: >
    删除路由表下自定义的路由
---


该功能用于删除路由表上的自定义路由。

**单个删除**

1. 在左侧导航栏，选择 **_"网络/基础网络/路由表"_** 菜单项，进入路由表页面。
2. 单击路由表名称项，进入路由表详情页面。
2. 单击“路由”页签，进入路由页面。
3. 单击路由右侧操作列 **_"删除"_** 按钮，弹出操作确认对话框。
4. 单击 **_"确定"_** 按钮，完成操作。


**批量删除**

1. 在左侧导航栏，选择 **_"网络/基础网络/路由表"_** 菜单项，进入路由表页面。
2. 单击路由表名称项，进入路由表详情页面。
2. 单击“路由”页签，进入路由页面。
3. 在路由列表中选择一条或多条路由，单击列表上方 **_"删除"_** 按钮，弹出操作确认对话框。
4. 单击 **_"确定"_** 按钮，完成操作。
