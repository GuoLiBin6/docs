description: >
    查看VPC对等连接的详细信息
---

1. 在左侧导航栏，选择 **_"网络/基础网络/VPC对等连接"_** 菜单项，进入VPC对等连接页面。
2. 单击VPC对等连接名称项，进入VPC对等连接详情页面。
2. 详情页面顶部菜单项支持对VPC对等连接进行删除操作。
3. 查看以下信息：
    - 基本信息：VPC对等连接的云上ID、ID、名称、状态、标签、域、平台、可用区、云账号、创建时间、更新时间、备注等。
    - 配置信息：包括本地VPC、对端VPC、对端账号等信息。
